# adminPortal

**adminPortal** is the administrative dashboard and management layer for the Financial System ecosystem. It allows operations teams and system administrators to oversee and control the various microservices, applications, and core systems.

## Features

- High-level overview of system health and metrics.
- Interface with the `finOS` APIs and other microservices for user and product management.
- Centralized administrative capabilities.

## How to Run

1. Navigate to the `adminPortal` directory.
2. Start the service using Docker Compose:
```bash
docker compose up -d --build
```
